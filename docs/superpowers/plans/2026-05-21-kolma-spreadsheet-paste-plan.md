# S41 엑셀 붙여넣기 입력 UX 계획

## 목표
- 연구자가 엑셀/스프레드시트에서 복사한 처방 행을 배합표에 바로 붙여넣을 수 있게 한다.
- 반복 수기 입력을 줄이고, 기존 처방 자료를 PoC 화면에 빠르게 옮길 수 있게 한다.

## 범위
- 배합표 입력칸에서 탭/줄바꿈이 포함된 붙여넣기 감지
- `원료명, 함량, 단위, 비율, 메모` 순서로 셀 값을 행에 반영
- 붙여넣은 행 수가 현재 행 수보다 많으면 행 자동 추가
- 붙여넣은 원료명은 최근 사용 원료 목록에도 반영
- 단위는 기존 선택지 `mg`, `g`, `%`, `ppm`만 반영하고 빈 값 또는 미지원 값은 기존 단위 유지

## 제외
- 파일 업로드 방식의 엑셀 import
- 서버 저장 또는 검증
- 원료 사전 매칭
- 단위 자동 변환

## 구현 순서
1. FormulaInputTable 실패 테스트 추가
2. 실패 테스트 확인
3. 붙여넣기 텍스트 파싱 및 행 반영 구현
4. 최근 사용 원료 저장 연동
5. 웹 검증
6. 한국어 커밋 메시지로 커밋 및 푸시

## 성공 기준
- 두 줄의 탭 구분 데이터를 원료명 칸에 붙여넣으면 두 개의 배합 행이 채워진다.
- 붙여넣은 행 수가 기존 행 수보다 많으면 새 행이 생긴다.
- 단위 선택지는 붙여넣은 `mg`, `g`, `%`, `ppm` 값을 반영한다.
- 붙여넣은 원료명은 최근 사용 원료 목록에 표시된다.
- 전체 웹 테스트와 빌드가 통과한다.
