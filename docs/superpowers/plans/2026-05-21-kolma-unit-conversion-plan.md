# S42 단위 변환 보조 입력 UX 계획

## 목표
- 연구자가 배합표에서 `g`와 `mg` 함량을 빠르게 변환할 수 있게 한다.
- 수기 계산 실수를 줄이고, 붙여넣은 처방 값을 화면에서 바로 정리할 수 있게 한다.

## 범위
- 배합표 각 행의 단위 칸에 `mg`, `g` 변환 버튼 제공
- `g -> mg` 변환 시 함량에 1000을 곱하고 단위를 `mg`로 변경
- `mg -> g` 변환 시 함량을 1000으로 나누고 단위를 `g`로 변경
- 빈 값 또는 숫자가 아닌 함량은 변환 버튼 비활성화
- `%`, `ppm`은 자동 변환 대상에서 제외

## 제외
- `%`, `ppm` 변환
- 일일 섭취량 기준 자동 계산
- 서버 저장 또는 검증
- 원료별 단위 권장값 자동 추천

## 성공 기준
- `1.5 g` 행에서 `mg` 변환을 누르면 `1500 mg`가 된다.
- `1500 mg` 행에서 `g` 변환을 누르면 `1.5 g`가 된다.
- 함량이 비어 있으면 변환 버튼이 비활성화된다.
- 전체 웹 테스트와 빌드가 통과한다.
