# S37 추천 근거 검색 링크 계획

## 목표
- AI 후보 Try 초안의 안전/규제 신호에서 관련 근거 검색으로 바로 이동할 수 있게 한다.
- 연구자가 신호를 본 뒤 원료명 기반 근거 후보를 빠르게 확인하도록 한다.

## 범위
- 대시보드 안전/규제 신호 카드에 `근거 검색` 링크 추가
- 링크 검색어는 신호의 `relatedIngredients`를 공백으로 연결해 구성
- 링크 대상은 `/knowledge?q=<검색어>` 형식
- API 응답 구조 변경 없이 웹 렌더링만 보강
- 로컬 fallback 신호에도 동일하게 적용

## 제외
- 추천 API의 근거 링크 필드 추가
- 검색 결과 자동 필터링 고도화
- 근거 상세 화면 추가
- 외부 API 실시간 조회

## 구현 순서
1. 웹 대시보드 실패 테스트 추가
2. 실패 테스트 확인
3. 대시보드 안전/규제 신호 카드에 근거 검색 링크 추가
4. 전체 웹 검증
5. 한국어 커밋 메시지로 커밋 및 푸시

## 진행 상태
- [x] 웹 실패 테스트 추가
- [x] 실패 테스트 확인
- [x] 웹 구현
- [x] 전체 웹 검증

## 성공 기준
- 안전/규제 신호 카드에 `근거 검색` 링크가 표시된다.
- 링크 href는 관련 원료명을 포함한 `/knowledge?q=...` 형식이다.
- 링크 접근성 이름에는 신호명이 포함된다.
- API 추천 신호와 로컬 fallback 신호 모두 같은 링크 규칙을 사용한다.
